1. Чтобы написать скрипт для файла, нужно на сервере зайти в свою папку
2. Вызвать текстовый редактор nano и сразу записать имя файла с разрешением sh: nano primer.sh
3. bash-скрипт начинается с команды #!/bin/bash, тем самым обозначая что файл со скриптом
4. Запишем скрипт:
   - в переменную var попадают єлементы из списка: for var in first second third fourth fifth  
   - пока не закончились єлементы, цикл будет делать do
   - выводить слово The и подставлять значения записанные в переменной var и после дописывать слово item
   - done выхол из цикла
 
#!/bin/bash
for var in first second third fourth fifth
do
echo The  $var item
done


5. После записи скрипта в файл нужно выйти с сохранением.
6. Нужно сделать файл исполняемым, с помощью ввода команды chmod +x ./имя файла
7. После этого уже запускаем исполение файла: ./имя файла
